Mundo Web

Desde la creación del primer sitio web en 1990 por Tim Berners-Lee se han producido muchos avances en el desarrollo web, actualmente existen varias formas en las que se pueden hacer, ya no necesitas dominar la programación necesariamente pues está la opción de crear webs con CMS (sistemas de gestión de contenidos) que permiten crear sitios web a partir de plantillas ya creadas por desarrolladores.

¿Qué es la web, sitio web y página web?

La web : Es la forma más sencilla y general para referirse a la World Wide Web (red de informática mundial) que es el sistema de documentos interconectados por enlaces de hipertexto, disponibles en internet.

Sitio web : Hace referencia a un conjunto de documentos en formato HTML (páginas web) que se encuentran relacionados con un dominio o subdominio en particular y único en la web.

Página web : Es un documento o fuente de información, en formato HTML y puede ser accesible desde un dispositivo físico, una intranet, o Internet.

En la práctica se suele usar mal estos términos muchas veces, por ejemplo es recurrente escuchar a las personas referirse a un sitio web como página web, esto es incorrecto a menos que el sitio web solo contenga una página web algo que es difícil que ocurra pues mínimo suelen tener 2 páginas web, una con el contenido del que trata el sitio web y otra con las politicas de privacidad y términos y condiciones.

¿Qué es lo más importante hoy en día?

Es hacer útil el sitio web que tengamos, la OPTIMIZACIÓN para los buscadores llamado SEO es lo más importante hoy y para el futuro, un sitio web no optimizada es como si no existiera en la internet para los usuarios.

La optimización para buscadores (SEO) es una combinación de buenas prácticas al construir un sitio web a nivel de código, en la forma en la cual distribuimos y expresamos el contenido (texto e imágenes) y aquellas estrategias de marketing digital que permiten decirle a google u otro navegador web de que somos relevantes para las keywords (palabras o conjunto de palabras buscados en internet) que se relacionan con nuestro sitio web.

Los buscadores como Google (el más importante) tienen como objetivo brindar el mejor contenido a los usuarios en internet es por ello que los creadores de sitios web deben de tomar en cuenta las recomendaciones técnicas que da e investigar cuales son los factores más importantes para ser más visibles y relevantes que los demás en nuestro respectivo sector o rubro.

¿Entonces si mi web ha sido creada con SEO tendrá éxito sobre los demás?

La respuesta es que tienes mayores probabilidades de logralo. Tendrás éxito si la optimización es constante y ofresces un buen contenido a los que ingresen a tu sitio web. Para ganar posicionamiento en las keywords tenemos que desarrollar todo el SEO , el técnico, On page y Off page, estos dos últimos con lleva la aplicación del marketing digital y diversas herramientas que nos permitiran ser mejores que los demás.

A continuación te presento contenido muy importante para entender como se construye los sitios web actualmente para optimizar costos(uso de CMS), entender y aplicar SEO, las formas de saber el comportamiento de los visitantes al sitio web (Google analytics), la publicidad de paga (SEM) y la importancia de saber un poco de los lenguajes con el cual se crean los sitios web (HTML, CSS Y JAVASCRIPT).

¿Cómo potenciar mi sitio web?

Si quieres que tu web tenga éxito necesitas de una estrategia que se adapte al contexto y sea dinámica frente a los cambios, para eso necesitas aplicar SEO, utilizar la api de facebook, la automatización de envíos de correos masivos bajo cierta segmentación, monitoreo del comportamiento de los que ingresan a tu web y la medición de los resultados de tus campañas. Si tu sitio web está en WordPress para hacer esto existen diversas herramientas de las cuales considero indispensable las siguientes siete: Google AdWords, Google Analytics, Google Search Console, api de facebook, Semrush versión gratuita, Yoast Seo versión gratuita y Contact Form 7 en caso no quieras pagar un servicio premium de MailPoet NewsLetter, Sumo, MailChimp u otro.

Tienda virtual

Se trata de un tipo de E-commerce, una web para vender que cuenta con uno o más métodos de pago vía internet, las pasarelas de pago suelen ser los más frecuentes para que los usuarios puedan pagar con tarjeta de crédito o débito.


¿Cómo obtengo una tienda virtual?

La forma más económica es usando un CMS pues nos permite construir y administrar la web sin saber de código. Eso no quiere decir que sea fácil y lo pueda hacer cualquiera, hay que aprender a usar el CMS y los complementos necesarios para potenciar la web, por ello es recomendable que contrates a un especialista para la creación y también te capacite para la administración de la misma. También puedes mandarla hacer a un desarrollador web pero la inversión será mucho mayor y en función al tamaño de la tienda virtual, pero será una web más ligera y 100% personalizada.


Beneficios:

Puedo vender productos sin la necesidad de tener un local físico.


Inconvenientes:

Existe responsabilidad legal si ocurren problemas en las operaciones o transacciones que se realizan en la tienda virtual y a medida que cresca será más costosa mantenerla optimizada y administrarla.

Web de Afiliación

Esta web contiene productos o servicios no propios que muestra a los visitantes de la web con el objetivo de inducirlos a la compra, cuando el usuario se traslada a la tienda virtual desde la web de afiliación y compra se estará ganando un porcentaje del precio de venta.


¿Cómo obtengo una web de afiliación?

La web la puedes hacer con un CMS como wordpress o si quieres que sea lo más ligera posible y personalizada puedes mandarlo hacer con un desarrollador web, pero requerirá una mayor inversión. Para la afiliación se requerirá cumplir los requisitos que te solicite la tienda virtual. Las tiendas virtuales a las cuales te puedes afiliar son Amazon, eBay , Linio , etc.


Beneficios:

Ganas dinero y no necesitas comprar ni fabricar los productos o servicios que promocionarás, estás fuera de responsabilidad legal de cualquier problema que pueda surgir en la transacción comercial que el usuario pueda realizar en la tienda virtual al cual lo dirijás.


Inconvenientes:

Tendrás que trabajar bastante en el SEO técnico y on page.

Web de servicios

Son para profesionales y empresas de servicios, que tienen como objetivo conseguir clientes de las búsquedas de los usuarios en google y servir de plataforma de aterrizaje para audiencias de redes sociales.


¿Cómo obtengo una web de servicios?

Estas web son las más sencillas de crear por lo general, puedes usar CMS como WordPress que fue creado especialmente para hacer blogs o páginas informativas o mandarla hacer a un desarrollador web.


Beneficios:

Brindar confianza y seguridad con el objetivo de conseguir clientes y fidelizarlos con contenido de valor.


Inconvenientes:

Tendrás que trabajar bastante en el SEO off page, estar constantemente aplicando estrategias de Branding.

Web de entretenimiento

Es una web que brinda exclusivamente contenido de valor al usuario, orientada al ocio, educación u otra cosa, algunas de estas web son blogs o webs de series, animes, etc.


¿Cómo creo una web de entretenimiento?

Puedes usar CMS como WordPress para ahorrar dinero o mandarla hacer a un desarrollador web.


Beneficios:

Estos sitios web generan gran cantidad de tráfico y por ello puedes ganar dinero por publicidad con Google o de manera independiente alquilando espacios de publicidad a otras webs.


Inconvenientes:

Tendrás que trabajar bastante en el SEO on page, el contenido de valor que ofrecerás será de vital importancia acompañada una buena experiencia de usuario y usabilidad (UX/UI).

¿Cuánto cuesta obtener un sitio web en Perú?

Los precios son bien variables que pueden ir desde los S./400 soles ($115) hasta más de 10 mil soles ($2850) pues dependerá del tamaño del sitio web, de quién realice el trabajo, y de las caracterisicas y especificaciones que se soliciten, por eso es importante informarse bien si quieres realizar una compra a un buen precio pero de calidad. Por lo general lo más barato te puede salir inserbible para tus objetivos y lo peor es que tal vez nisiquiera te darás cuenta de ello por ignorancia.

¿Qué debo saber antes adquirir un sitio web?

Para saber si el precio que te ofrece el desarrollador web o empresa de desarrollo está justificado tienes que saber sobre dos aspectos importantes, como será hecho tu sitio web y las caracteristicas del hosting donde será alojado para poder estar en internet.

La manera en que se va crear tu sitio web

A partir de una plantilla: El desarrollador compra una plantilla con las funcionalidades y diseño que se acerquen a los que necesita tu sitio web, luego hace las modificaciones necesarias, muchas veces insertando código para inabilitar ciertas cosas y otras líneas de código para implementar otros estilos o funcionalidades , al hacer estas modificaciones esta creando una plantilla hijo que usa las funcionalidades y diseño del padre para reducir el trabajo en la construcción del sitio web. En esta modalidad es muy importante que se use una buena plantilla (si no hizo modificaciones en el código) o plantilla padre (si realizo modificaciones en el código). Saber cuales son las mejores plantillas es fácil investigando con Google, una de las que más reputación de calidad tiene es Divi, y otra que me parece buena a mi es Generatepress.

A código sin usar las plantillas: El desarrollador primero realiza un diseño visual y dinámico de todo el sitio web para que el cliente pueda apreciarlo y dar las observaciones o indicaciones de cambios, una vez aprobado el diseño el desarrollador comienza la construcción del sitio web. Esta modalidad es la mejor pero es la más costosa pues se trata de una web hecha desde el comienzo para ti.

Las caracteristicas del Hosting donde estará tu sitio web

Espacio en disco SSD : Dependiendo del tamaño de tu sitio necesitarás mayor espacio, un blog personal no tendrá problema por lo general con esto pero una tienda virtual si puede presentar problemas de este tipo, por ello si tu proyecto es serio debes consultar a un desarrollador sobre esto.

Certificado SSL: Secure Sockets Layer, es un protocolo de seguridad que usa una moderna encriptación para enviar y recibir información sensible en todo internet, trabaja al crear un canal seguro entre el navegador y el servidor del sitio web al que el usuario quiere entrar o conectarse por ello los navegadores como Google lo consideran muy importantes y es el motivo por el cuál nuestro sitio web debe tenerlo como parte de la optimización SEO. El hosting valdrá más si lo incluye en el precio.

Transferencia mensual: Es la cantidad de datos que registra el servidor al cargar la página web durante un mes , los MB que se registrará serán porporcionales a la cantidad de visitas del mes al sitio. Si crees que tu sitio web tendrá bastante tráfico necesitarás evaluar bien esta caracteristica ofrecida por tus opciones de hosting a contratar.

Reputación en el mercado: Es tal vez la más importante pues te garantiza que dan un buen servicio, para saber sobre la reputación revisa varios blogs y foros.

¿Qué es un desarrollador web?

Es una persona que construye sitios web utilizando código, es decir maneja el lenguaje de marcado HTML, lenguaje de estilos CSS y al menos un lenguaje de programación que puede ser Javascript, Python, etc. Es importante que sepas que los desarrolladores web tienen dos especializaciones, la que trabaja la parte visual de la web y las interaciones básicas con el usuario llamada FrontEnd y la que trabaja la interacción de la base de datos con los servidores llamada Backend.

¿Para construir una web necesito a los dos especialistas (FrontEnd y Backend)?

Depende, si se trata de una web sencilla cualquiera de los dos especialistas podría realizarla pues ambos conocen un poco del otro. Si se trata de una web grande y compleja se necesitaran de ambos especialistas y adicionalmente otros como un diseñador web enfocado a la experiencia de usuario.

NOTA: Es importante mencionar que el término diseñador web no es igual a desarrollador web pero se suelen usar como equivalentes en ocasiones, un diseñador web puede ser un diseñador gráfico especializado en UX-UI que no requiere tener todos los conocimientos de un FrontEnd. Para proyectos pequeños y medianos por lo general el FrontEnd hace de diseñador y desarrollador.
