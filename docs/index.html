Mundo Web

En 1990, se creó el primer sitio web por Tim Berners- Lee quien, a partir de esto, estableció las bases para el desarrollo y estudio progresivo de dichas plataformas virtuales por empresas de programación y sistemas.

En la actualidad existen diversos modos de crear un sitio web porque nuevas tecnologías surgieron ante las necesidades del mercado, incluso no hace falta saber de programación para crear uno. No te confundas todo lo que hay en la web es código y programación, lo que sucede es que se crearon programas informáticos como los CMS (Sistemas de Gestión de Contenidos) que permite crear sitios web con plantillas ya creadas por desarrolladores de esta manera no tienes que escribir código sino solo introducir el contenido (texto, imágenes, videos) que quieres que vea tu audiencia.

¿Qué es la web, sitio web y página web?

La web : significa Red de Informática Mundial o World Wide Web (www), es decir, es un sistema de documentos de hipertexto o hipermedia interconectados y accesibles a través de Internet. Al igual que una telaraña todo está conectado con todo.

Sitio web : es un conjunto de documentos electrónicos (hipertexto) relacionados con una dirección IP, de manera más simple se puede definir como un conjunto de páginas web conectados a un dominio único y particular en la web.

Página web : o página electrónica, página digital o ciber-página, es un documento electrónico que puede contener texto, imágenes, sonidos, videos, enlaces y muchas otras cosas, por lo general en formato HTML o XHTML. Se puede tener acceso a ella desde un dispositivo físico (computadoras, celulares, table, etc), una intranet (red informática interna) o internet.

Es común el mal uso de esos términos, por ejemplo, se suele confundir página web o web para referirse a un sitio web.

Hoy ¿qué es lo más importante?

La optimización, utilidad, inmediatez y la conexión emocional con el visitante del sitio web, al cumplir estos requisitos podemos lograr éxito.

La optimización: al tratarse de sitios web a los cuales tenemos acceso a través de internet utilizando navegadores o buscadores como Google, Bing y FireFox, tenemos que optimizar nuestro sitio web para que estos búscadores lo encuentren fácilmente, lo entiendan y lo muestren entre las primeras opciones de respuesta a los usuarios de nuestro interés. Esta optimización es llamada SEO (Search Engine Optimization o Motores).

La utilidad e inmediatez: los creadores tienen muy en cuenta estas dos palabras porque, la primera, utilidad, es la capacidad que tiene un contenido o servicio de ser aprovechado al máximo; la segunda, inmediatez, es una respuesta rápida a la solución. Aplicando ambas obtenemos buena calidad en el contenido y funcionalidad del sitio web, estas caracteristicas se encuentran dentro del UX-UI (User Experience-User Interface).

La conexión emocional: la logramos a través de un mensaje con una redacción, colores e imágenes que conecten con el visitante de nuestro sitio web, esto nos sirve para influenciar a realizar determinadas acciones, que pueden ser suscribirse a un boletín, hacer una compra, inscribirse a un webinar, etc. Todo esto se estudia en el copywriting que viene a ser un conjunto de técnicas en combinación con creatividad para producir textos persuasivos en beneficio al Marketing y a las Ventas.

Entonces, si mi web fue creada con SEO, UI-UX y Copywriting … ¿Tendrá éxito sobre las demás?

La respuesta es Sí. El éxito se crea si la optimización es constante y la calidad de contenido es buena para que los consumidores ingresen a tu sitio web.

Para ganar posicionamiento en las keywords (palabras clave que buscan los usuarios) de nuestro interés se debe aplicar el SEO On Page y Off Page, el primero hace referencia al cumplimiento de las recomendaciones que nos da Google para la construcción de un sitio web, entonces el SEO On page son acciones que podemos realizar internamente en nuestro sitio web con el objetivo de ser más visibles e importantes para el buscador, el segundo hace referencia a las acciones externas que brindan un impacto positivo hacia nuestro sitio web, estas acciones externas tienen relación con la autoridad que tenemos en el sector y se manifiesta cuando se hace una mención a nuestro sitio web desde otra.

¿Qué tipos de sitios web existen?

Existen varias formas de clasificar a los sitios web, ya sea por su contenido o beneficios, pero la más importante es la que está en sintonía con los negocios a los cuales se rige.

Haz click para que muestre el contenido.

Tienda virtual

Básicamente, es un e-commerce o Comercio Electrónico En Línea.
Posee varios métodos de pago vía transferencia, ya sea, con tarjeta de crédito, débito o apps bancarias (PayPal, Mercado Pago, entre otras).

¿Cómo obtengo una tienda virtual?

La forma económica y práctica es usando un CMS porque permite construir y administrar la web sin saber de códigos.

Tiene sus dificultades este tipo de programas y hay que aprender a usarlo, pero con una buena guía es suficiente. Si no estamos conformes con el resultado que obtuvimos, hay que contratar a un técnico para la creación de la tienda virtual y para la capacitación de la misma.

Por otro lado, también se puede contratar a un desarrollador web, pero la inversión será mayor en función al tamaño de la tienda, pero, dando como resultado, una página ligera y personalizada.

Beneficios:
Vender productos sin poseer un local físico.

Inconvenientes:
Responsabilidad legal en operaciones y transacciones.
A medida que crezca la web será costosa de optimizar y administrar.

Web de afiliación

Es una web de enlaces o web de promoción, es decir, es un sistema mediante el cual se colocan enlaces a productos o servicios de otras páginas webs o tiendas de compra. Si se realiza una compra o venta, recibe una comisión por ella (dependiendo del precio del producto).

¿Cómo obtengo una web de afiliación?

Se puede crear con un CMS como WordPress o comprar el servicio a un desarrollador para que optimice y cree la página web.

Para afiliarte a este tipo de webs se necesitan ciertos requisitos a cumplir. Tal es el caso de: Amazon, eBay, Linio, etc.

Beneficios:

Ganar dinero sin comprar ni fabricar productos o servicios.
Alejado de toda responsabilidad legal.
No hay problemas con transacciones o pagos comerciales.
A mayores afiliados, mayor ingreso.
A mayor precio de producto o venta, mayor comisión.

Inconvenientes:
Trabajo constante de SEO técnico y SEO On Page.

Web de entretenimiento

Son web o espacios dedicados al entretenimiento y diversión del usuario, ya sea, para escuchar música, juegos online, ver películas, videos, apuestas, etc.

¿Cómo obtengo una web de entretenimiento?

Se puede crear mediante el uso de CMS como WordPress o contratando un programador.

Beneficios:

Gran cantidad de entrada de usuarios.
Ganar dinero por publicidad de Google.
Ganar dinero alquilando espacios de publicidad a otras webs.
Ganar dinero como sitio web de afiliación.

Inconvenientes:

Trabajo constante en SEO On Page.
Trabajo de reclamos y asistencia al usuario ante malas experiencias.
Acompañamiento y usabilidad del sitio web (UX/UI) hacia los consumidores.

Web de servicios

Son las más utilizadas por profesionales y empresas porque se especializan en conseguir clientes de los usuarios de Google, además, sirven como plataforma “imán” hacia las redes sociales y volverse tendencia con el tiempo.

¿Cómo obtengo una web de servicios?

Se puede crear mediante CMS como WordPress, ya que, tiene formato y estructura de blog. Al ser páginas informativas son sencillas de crear y programar.

Beneficios:

Atraer clientes.
Promulgar confianza y seguridad de los productos a los futuros clientes.
Lealtad hacia el servicio como contenido de valor.

Inconvenientes:

Trabajo constante en SEO ON Page.
Uso intensivo de Identificación Comercial o Branding.
Verificación de análisis comerciales.
Renovación de diseños y estrategias comerciales digitales.

¿Cuál es el precio de un sitio web en Perú?

Los precios van desde los 400 soles ($115 dólares) hasta más de 10 mill soles ($2850 dólares). Esto depende de quién realice el trabajo o desarrollador, el tamaño del sitio web y las especificaciones que soliciten (características específicas, personalización, animaciones, enlaces, contenido audiovisual, etc).

Es necesario estar informado sobre costos totales de la página web, por un lado, para realizar una buena compra y, por otro lado, porque los precios fluctúan según la demanda y la moneda.

¿Qué debo saber antes de adquirir un sitio web?

Antes que nada, el precio que te ofrece el desarrollador o la empresa de desarrollo tiene que ser acorde al costo de creación del sitio web y a las características del hosting.

¿Qué quiere decir esto? Fácil, el precio de desarrollo personalizado y el alojamiento web para almacenar cualquier contenido accesible vía web.

Formas de crear un sitio web

Con una plantilla.

El desarrollador se basa en el uso de una plantilla o página web prediseñada o template para el diseño y funcionalidades del sitio web deseado por el comprador. A partir de modificaciones, uso de códigos para inhabilitar o implementar estilos y diseños va construyendo la misma.

Es importante que se use una plantilla de buena calidad para reducir el trabajo en la construcción del sitio web y como base sustentable para otras modificaciones como texto, fotos, logotipos, colores e información de contacto.

Las mejores plantillas y con mejor puntuación son:
Divi (es una mega plantilla en WordPress que permite crear cualquier diseño).
Generatepress.

Con código, sin usar plantillas

El desarrollador inicia con un borrador sobre el diseño visual del sitio web del cliente para que este pueda apreciarlo. Luego de las observaciones e indicaciones estimadas por el comprador, se realizan los cambios y el desarrollo del sitio web.
Considero esta modalidad como la más apropiada porque se trata de una web hecha desde cero, además, que la paga va a ser mayor.

Características del hosting

Espacio SSD.

SSD es el acrónimo de Solid State Drive o Unidad en Estado Sólido. El hosting SSD es un servicio de alojamiento de información de páginas web que utiliza dispositivos de almacenamiento de información en estado sólido.
Al depender del tamaño del sitio web se necesita mayor o menos espacio. Por ejemplo, un blog personal no ocupa gran cantidad de almacenamiento, pero, una tienda virtual requiere mayor consumo de memoria y causando problemas de optimización.

Certificado SSL.

SSl quiere decir Secure Sockets Layer o Certificado Digital, es decir, es un protocolo de seguridad que se encarga de autenticar la identidad del sitio web y cifra la información transmitida, para crear un canal seguro entre el navegador, el usuario y el servidor del sitio web.
Es importante tener este Certificado porque el servidor de Google lo requiere, además, es parte de la optimización SEO y, lo mejor, el hosting valdrá más si lo incluye en el precio.

Reputación en el mercado.

La reputación es la percepción que tienen los consumidores sobre un determinado producto o servicio.
Es importante evaluar las opiniones e impresiones positivas y negativas del servicio para hacer las correcciones necesarias, además, permite crear campañas que tengan un mayor impacto en los consumidores.

¿Qué significa ser un desarrollador web?

Es una persona que construye sitios web utilizando código, quiere decir, que maneja el lenguaje HTML, lenguaje de estilo CSS y, al menos, un lenguaje de programación como JavaScript, Python, etc.

Es importante saber que los desarrolladores tienen dos especializaciones:

FrontEnd: trabaja la parte visual de la web y las interacciones básicas con el usuario.
Backend: trabaja la interacción de la base de datos con los servidores.

Para construir una web ¿Necesito a un especialista en FrontEnd y otro en BackEnd?

Si se trata de una web sencilla cualquiera de los dos especialistas puede realizarla porque ambos comparten parte del conocimiento. En cambio, si la web es más grande y compleja es necesario contratar a los dos especialistas y, además, a un diseñador web capacitado en la experiencia del usuario.

Nota:Es importante mencionar que el término diseñador web no es lo mismo a desarrollador web.
Un diseñador web se basa en el diseño gráfico especializado en UX-UI y no requiere los conocimientos de un FrontEnd. En cambio, para pequeños y medianos proyectos el especialista en FrontEnd hace de diseñador y desarrollador.
